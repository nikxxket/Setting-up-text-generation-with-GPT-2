Настройка генерации текста с помощью GPT-2

Простой и мощный интерфейс для генерации текста с использованием модели GPT-2 и библиотеки Hugging Face Transformers. 
Скрипт поддерживает две стратегии декодирования: жадный поиск и семплирование с настраиваемыми параметрами.

Возможности:

Два метода генерации:
1) Жадный поиск (Greedy Search): Детерминированная генерация, при которой на каждом шаге выбирается наиболее вероятный следующий токен;
2) Семплирование (Sampling): Стохастическая генерация с настраиваемыми параметрами температуры и top-p для получения более разнообразных и креативных результатов.

Интерактивный интерфейс: Простой ввод через командную строку для указания промпта и выбора параметров генерации.

Параметры генерации:
1) Жадный поиск (Gready Search) - детерминированный результат (подходит для задач, где важна предсказуемость и надежность);
2) Семплирование (Sampling) - случайный выбор токена.

Параметры:
1) Temperature: Контролирует "креативность" модели: 0.1-0.3 консервативный ответ, 0.7-0.9 баланс между креативностью и связанностью текста, 1.5-2.0 хаотичный текст;
2) Top-p: Ограничивает выбор токенов теми, чья совокупная вероятность не превышает указанное значение: 0.1-0.3 высокая детерминированность, 0.5-0.7 баланс, 0.9-1 креавтивность (высокий риск галлюцинаций).
